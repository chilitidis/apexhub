# Design notes — Market News & Pre-Market Briefing (from user screenshots)

Note: screenshots show a light theme variant, but the app default is the dark navy theme.
Build BOTH pages in the existing dark navy theme (#070F1C bg, #0D1E35 cards, gold #F4A261 accents,
profit #00897B/#00C2A0, loss #E94F37, muted #4A6080, Space Grotesk + mono) — matching PatternAnalysisPage.

## Market News page ("High Impact Economic Events")
- Header: newspaper icon + title "High Impact Economic Events" (Greek context but title kept EN per screenshot).
- Subtitle: "This week's high-impact events from Forex Factory"
- Top-right pills: [High Only] (active, red dot) | [All Impact] (yellow dot) toggle, plus [Refresh] button (icon).
- Meta line: clock "Updated 02:55 AM · Source: Forex Factory"
- Events grouped by DAY. Each day group:
  - Day header chip: "MONDAY, JUN 1" + right-aligned "N events" count, with a faint divider line.
  - Event card (rounded, card bg, subtle border):
    - Left: flag/currency badge (we'll use currency code badge colored by impact since no flag assets) with currency code under it (e.g. USD).
    - Title (bold) + impact badge ("🔴 HIGH" red pill / medium yellow).
    - Below title: "Forecast: X   Previous: Y" (muted, values bold).
    - Right: clock + "Mon, Jun 1 at 17:00" local time + "Released" tag (grey) if event already passed.
- Footer: "All times shown in your local timezone · Data from Forex Factory"
- Filter default: High Only. All Impact shows High+Medium (and low). Keep it simple: "High Only" vs "All Impact".

## Pre-Market Briefing page
- Header: sunrise icon + title "Pre-Market Briefing" + calendar icon + date label "Σάββατο 6 Ιουνίου 2026".
- Top-right: [Ανανέωση] button (gold) / shows [Δημιουργία...] with spinner while generating.
- Body: single big card. Header row: "ΗΜΕΡΉΣΙΑ ΑΝΆΛΥΣΗ" (small label) + time "02:54 πμ" right.
- Content = markdown rendered via <Streamdown>:
  ## 🌅 Pre-Market Briefing — Παρασκευή 5 Ιουνίου 2026
  ### 📊 Market Sentiment (paragraph)
  ### 🔴 High Impact Events Today (bullets: **HH:MM UTC**: CCY - title - comment)
  ### 💱 Key Pairs to Watch (bullets: **PAIR**: reason)
  ### 📈 Bias Summary (table: Pair | Bias | Reason)
  ### ⚠️ Risk Warnings (paragraph)
  ### 💡 Mindset Reminder (paragraph)
  *Briefing generated by Ultimate Trading Journal AI | For educational purposes only | Not financial advice*
- Loading card: centered spinner + "Αναλύω αγορά και events... (~10 δευτερόλεπτα)"
- Briefing uses today's High+Medium events, time formatted "HH:MM UTC".
