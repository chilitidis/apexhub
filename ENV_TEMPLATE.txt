# CLERK AUTH (recommended for Railway / self-hosted production)
# -----------------------------------------------------------------------------
# APEXHUB ships with Clerk as the primary multi-tenant auth provider. Every
# visitor signs up with email or Google, gets their own empty journal scoped
# to their `users.id`, and never sees another user's trades. When BOTH of
# these keys are set, Clerk automatically takes precedence over Manus OAuth
# and any DEMO_MODE flag below is ignored.
#
# Get them from https://dashboard.clerk.com → API Keys:
#   - Publishable key  → starts with `pk_test_` or `pk_live_`
#   - Secret key       → starts with `sk_test_` or `sk_live_`
VITE_CLERK_PUBLISHABLE_KEY=pk_test_replace-me
CLERK_SECRET_KEY=sk_test_replace-me
